Description

This is a demo website for browsing fonts with Google Fonts service, producing some features:
  • Search with keyword
  • Category filter
  • Sorting
  • Editable sample text
  • Custom font color
  • Custom font size
Develop with Angular framework, W3.CSS and Font Awesome.
Build with Angular CLI, then hosted on the Heroku.
